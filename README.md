# SilentProximity

**SilentProxmity** is a multiplayer first person project in a similar vein as popular games like REPO or Lethal Company. 
However as opposed to those projects SilentProximity does not use voice chat and instead players are only able to communicate non-verbally. 
Using the OpenCV python library users webcam information is stored as json files. The python files are run as a websocket in the Godot Game Engine which then uses the steamwork
plugin to allow sending the webcam display to all players. This allows all players to see eachothers webcam footage.
